ol (Orderde List) : 순서가 있는 목록

 
        type속성 : 표시되는 숫자 모양을 변경

        type="a" : 영어 소문자 (a,b,c,d,e.....)
        type="A" : 영어 대문자 (A,B,C,D.....  )
        type="i" : 로마 소문자 (i, ii, iii....)
        type="I" : 로마 대문자 (I,II,III.....)

        start 속성 : 시작 숫자 지정
        reversed : 숫자 역순으로
  1. 컴퓨터
  2. 샤프
  3. 지우개
  4. 계산기

dl 태그(Description List) : 설명 목록

            dl 태그 : 설명 목록이 작성될 판(공간)
            dt 태그 : 설명 제목
            dd 태그 : 설명 내용
